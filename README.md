# 🔢 Count Number of Digits

A simple web application that counts the number of digits in a user-entered number using HTML and JavaScript.

The user enters a number in the input field and clicks the **Count** button. The program then calculates and displays the total number of digits instantly on the webpage.

---

## 🚀 Features

- Simple and easy-to-use interface
- Accepts numeric input from the user
- Instantly counts digits using JavaScript
- Displays result dynamically on the webpage
- Beginner-friendly project

---

## 🛠 Technologies Used

- **HTML5** – For webpage structure  
- **JavaScript** – For logic and DOM manipulation

---

## 📂 Project Structure

```
count-digits
│
└── index.html
```

---

## ⚙️ How It Works

1. The user enters a number in the input field.
2. When the **Count** button is clicked, the `countDigits()` function runs.
3. JavaScript reads the input value.
4. The program calculates the length of the number using `.length`.
5. The result is displayed on the webpage.

---

1. Clone the repository
